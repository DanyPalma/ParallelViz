# ParallelViz
Parallel Fast Fourier Transform: A High-Performance Rust Implementation for Spectral Analysis

## To compile
`cargo build --release`

## To run
`cargo run -- audio/<name>.wav`

FILES MUST BE .WAV!!! 

# convert from .mp3 to .wav using ffmpeg

`ffmpeg -i <oldname>.mp3 <newname>.wav`
